<h1>Wireless Mobile Computing & Mobile App Technology</h1>

<p>
Dokumen ini menjelaskan konsep <b>Wireless Mobile Computing</b> serta teknologi pengembangan aplikasi mobile 
(Native, Hybrid, dan Progressive Web App) menggunakan framework <b>PREP (Point, Reason, Example, Point)</b>.
</p>

<hr>

<h2>1. Wireless Mobile Computing</h2>

<h3>Point</h3>
<p>
Wireless Mobile Computing adalah teknologi yang memungkinkan perangkat mobile seperti smartphone, tablet, 
dan laptop untuk berkomunikasi dan mengakses data tanpa menggunakan kabel melalui jaringan nirkabel.
</p>

<h3>Reason</h3>
<ul>
    <li>Memberikan mobilitas tinggi (akses data kapan saja dan di mana saja)</li>
    <li>Mendukung aplikasi real-time seperti chat, GPS, dan transaksi online</li>
    <li>Menghubungkan perangkat dengan internet dan cloud computing</li>
</ul>

<p>Komponen utama:</p>
<ul>
    <li><b>Perangkat:</b> Smartphone, tablet, laptop</li>
    <li><b>Jaringan:</b> WiFi, 4G, 5G</li>
    <li><b>Aplikasi:</b> Mobile apps dan web apps</li>
</ul>

<h3>Example</h3>
<ul>
    <li>Menggunakan Google Maps untuk navigasi</li>
    <li>Aplikasi kasir berbasis cloud</li>
    <li>Chat WhatsApp secara real-time</li>
</ul>

<h3>Point (Kesimpulan)</h3>
<p>
Wireless Mobile Computing merupakan dasar dari teknologi mobile modern karena memungkinkan konektivitas fleksibel dan real-time.
</p>

<hr>

<h2>2. Teknologi Mobile Application</h2>

<h3>A. Native App</h3>

<h4>Point</h4>
<p>
Native app adalah aplikasi yang dikembangkan khusus untuk satu platform seperti Android atau iOS.
</p>

<h4>Reason</h4>
<ul>
    <li>Akses penuh ke hardware (kamera, GPS, NFC)</li>
    <li>Performa tinggi dan stabil</li>
    <li>User Interface lebih responsif</li>
</ul>

<h4>Example</h4>
<ul>
    <li>Aplikasi mobile banking</li>
    <li>Game mobile</li>
    <li>Aplikasi kamera</li>
</ul>

<h4>Point</h4>
<p>
Native app cocok untuk aplikasi yang membutuhkan performa tinggi dan akses hardware secara maksimal.
</p>

<hr>

<h3>B. Hybrid App</h3>

<h4>Point</h4>
<p>
Hybrid app adalah aplikasi yang dikembangkan menggunakan satu codebase tetapi dapat dijalankan di berbagai platform.
</p>

<h4>Reason</h4>
<ul>
    <li>Pengembangan lebih cepat</li>
    <li>Satu kode untuk Android dan iOS</li>
    <li>Dapat mengakses hardware menggunakan plugin</li>
</ul>

<h4>Example</h4>
<ul>
    <li>Aplikasi e-commerce</li>
    <li>Aplikasi kampus</li>
    <li>Aplikasi kasir mobile</li>
</ul>

<h4>Point</h4>
<p>
Hybrid app cocok untuk efisiensi biaya dan waktu pengembangan dengan performa yang cukup baik.
</p>

<hr>

<h3>C. Progressive Web App (PWA)</h3>

<h4>Point</h4>
<p>
PWA adalah aplikasi berbasis web yang dapat berjalan seperti aplikasi mobile tanpa harus diunduh dari app store.
</p>

<h4>Reason</h4>
<ul>
    <li>Tidak perlu instalasi melalui store</li>
    <li>Dapat diakses melalui browser</li>
    <li>Lebih ringan dan cepat dikembangkan</li>
</ul>

<h4>Example</h4>
<ul>
    <li>Website toko online</li>
    <li>Web kasir berbasis browser</li>
    <li>Portal kampus</li>
</ul>

<h4>Point</h4>
<p>
PWA cocok untuk aplikasi ringan yang membutuhkan akses cepat dan kemudahan distribusi.
</p>

<h2>4. Kesimpulan</h2>

<h3>Point</h3>
<p>
Teknologi mobile application merupakan bagian penting dari Wireless Mobile Computing.
</p>

<h3>Reason</h3>
<p>
Setiap jenis aplikasi memiliki kelebihan masing-masing tergantung kebutuhan sistem.
</p>

<h3>Example</h3>
<ul>
    <li>Aplikasi kasir → Hybrid atau PWA</li>
    <li>Game mobile → Native</li>
    <li>Website kampus → PWA</li>
</ul>

<h3>Point</h3>
<p>
Pemilihan teknologi harus disesuaikan dengan kebutuhan aplikasi, performa yang diinginkan, serta sumber daya pengembangan.
</p>
