<h2>WMC (Web Mobile Computing)</h2>
WMC adalah konsep pengembangan aplikasi yang dapat diakses melalui perangkat mobile menggunakan teknologi web maupun aplikasi native.<br>
Tujuannya adalah memberikan akses layanan secara fleksibel, real-time, dan mudah digunakan di berbagai perangkat seperti smartphone dan tablet.<br><br>

<h2>Teknologi Mobile App</h2>
Terdapat 3 pendekatan utama dalam pengembangan mobile app:<br><br>

<h2>1. Native App</h2>
Aplikasi dibuat khusus untuk platform tertentu (Android/iOS).<br>
Kelebihan: performa tinggi, akses penuh ke hardware (kamera, NFC, GPS).<br>
Kekurangan: biaya tinggi karena harus buat versi berbeda.<br><br>

<h2>2. Hybrid App</h2>
Gabungan web dan native menggunakan framework seperti Cordova atau React Native.<br>
Kelebihan: satu kode untuk banyak platform.<br>
Kekurangan: performa di bawah native.<br><br>

<h2>3. PWA (Progressive Web App)</h2>
Aplikasi web yang terasa seperti aplikasi native.<br>
Kelebihan: ringan, tidak perlu install dari store, bisa offline.<br>
Kekurangan: akses hardware terbatas.<br><br>

<h2>FW PREP (Framework Preparation)</h2>
FW PREP adalah tahap persiapan framework sebelum pengembangan aplikasi.<br>
Meliputi:<br>
- Pemilihan framework (React, Angular, Vue, dll)<br>
- Setup environment (Node.js, tools development)<br>
- Konfigurasi API dan backend<br>
- Struktur project dan arsitektur aplikasi<br><br>

<h2>Kesimpulan</h2>
WMC memanfaatkan teknologi mobile (Native, Hybrid, PWA) untuk membangun aplikasi modern.<br>
FW PREP berperan penting agar proses development lebih terstruktur, cepat, dan efisien.<br>
